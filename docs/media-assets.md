## Compression Tools

- **Images**: [TinyPNG](https://tinypng.com) or [Squoosh](https://squoosh.app)
- **Videos**: [HandBrake](https://handbrake.fr)

## Naming Convention

- All lowercase
- No spaces (use hyphens if needed)
- Duplicate categories get numbered suffixes: `decoration.jpg`, `decoration2.jpg`, `decoration3.jpg`
